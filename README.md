# farkler

A Program to test the statistical success of various of farkle strategies.

Wrote in Java bc I like my OOPsies to be coffee flavored.

Handwritten, artisanal spaghetti code with a side of AI slop.

## Commands

```bash
mvnw clean              # delete target/
mvnw compile            # compile source
mvnw test               # run tests
mvnw package            # build JAR
mvnw exec:java          # run Farkle
mvnw clean compile exec:java  # clean build and run
```
